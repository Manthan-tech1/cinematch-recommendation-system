01
Content-based filtering
Builds a TF-IDF feature vector for each movie from genres, cast, director, keywords, and mood tags. Computes cosine similarity between the query movie and all others in the dataset.
02
Collaborative filtering
Uses a simulated user-item rating matrix. Finds users who rated the query movie highly, then computes Pearson correlation to identify similar users and surfaces what they watched next.
03
Hybrid scoring
Combines both scores with configurable weights (default 55% content + 45% collaborative). The final hybrid score is normalised to 0–100 and used to rank recommendations.
04
Diversity boosting
A diversity penalty is applied to avoid recommending 5 movies from the same genre. Results are re-ranked to balance relevance with variety across genre, era, and mood.
Algorithm formula
Hybrid scoring equation used in this system
// Step 1: Content-based score (cosine similarity) content_score = dot(vec_A, vec_B) / (|vec_A| × |vec_B|) // Step 2: Collaborative score (Pearson correlation) collab_score = Σ[(r_u,i - r̄_u)(r_u,j - r̄_u)] / √[Σ(r_u,i - r̄_u)² × Σ(r_u,j - r̄_u)²] // Step 3: Hybrid final score (weighted combination) hybrid_score = (0.55 × content_score) + (0.45 × collab_score) // Step 4: Diversity re-ranking final_score = hybrid_score × (1 - 0.15 × genre_overlap_penalty)

About this project

CineMatch is a hybrid movie recommendation system built as a data science portfolio project. It demonstrates two core recommendation paradigms — content-based and collaborative filtering — combined into a weighted hybrid model that outperforms either approach alone.

The dataset contains 60 curated movies across 8 genres with feature vectors encoding genre, cast similarity, director, mood, era, and popularity. The simulated user-item rating matrix contains 200 synthetic users with rating patterns modeled on realistic viewing behaviour (long-tail distribution, genre affinity clusters).

Key concepts demonstrated

TF-IDF vectorisation · Cosine similarity · Pearson correlation · Weighted ensemble scoring · Diversity re-ranking · Cold-start handling · Matrix factorisation concepts
